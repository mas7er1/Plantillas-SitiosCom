General Information

Software requirements

  • visual studio code used to edit the HTML,CSS and JS files.

The following scheme displays the Template files structure.

  • Documentation - contains documentation on template cusotmisation and support
  • Template - contains the main files of the template. which will be uploaded on server.
    • CSS - contains all Stylesheets used for UI.
    • images - contains all the images.
    • JS - contains all javascript library or plugin files.
    • index.html - contains all UI code.
    • home-rtl.html - contains all UI code for home rtl version.
    • blog.html - contains blog section code.
    • single-blog.html - contains single-blog page code.
    • 404.html - contains 404 page code.
    • portfolio.html - contains portfolio section code.
    • single-work.html - contains single portfolio section code.
    • services.html - contains services section code.
    • about1.html - contains about1 us section code.
    • about2.html - contains about2 us section code.
    • contact.html - contains contact UI code.
    • FAQs.html - contains FAQs UI code.
    • privacy-policy.html - contains privacy policy UI code.
    • terms-of-use.html - contains terms of use UI code.
    • contact.php - contains PHP contact form
